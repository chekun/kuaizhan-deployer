# 快站页面代码部署助手

#### How to use

```
❯ kuaizhan-deploy -h
Usage of kuaizhan-deploy:
  -appkey string
        kuaizhan app key
  -appsecret string
        kuaizhan app secret
  -file string
        path to source file
  -page string
        kuaizhan page id, optional
  -site string
        kuaizhan site id
```
